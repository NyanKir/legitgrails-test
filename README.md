## Quick start
Node version: v20.10.0

To switch between different version of node.js. You can use the utility:
https://github.com/nvm-sh/nvm
```bash
$ nvm use
$ npm install
```
Create `.env` file, from example `.env.template`.

## Scripts and how to run

```bash
# watch mode
$ npm run dev
```
